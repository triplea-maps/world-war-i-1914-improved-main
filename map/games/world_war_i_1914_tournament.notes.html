Based on the board game "Axis & Allies: WWI 1914"

Made by VictoryFirst

You can check out the Tournament rules here:
https://www.axisandallies.org/forums/assets/uploads/files/1564089791466-larry-harris-axis-amp-allies-wwi-1914-tournament-rules.pdf

Still work in progress. Not all features are governed by the engine, some require enforcement by the player. Use edit when needed.

Things from the Tournament Rules that have been added:
  • Setup changes.
  • Increased movement for land units (though needs player enforcement for some cases).
  • Port gives +1 movement.

All movement of land units has been set to 2, however there are some cases where the units can only move 1. Check the rules to see when that is the case.
IMPORTANT: The economic/political collapse rules have not been incorporated. Use edit to follow these rules.
